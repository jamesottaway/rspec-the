# RSpec::The

Easily make assertions about the contents of your `let` blocks

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'rspec-the'
```

And then execute:

```
bundle install
```

Or install it yourself as:

```
gem install rspec-the
```
